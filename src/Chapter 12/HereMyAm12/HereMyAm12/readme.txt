To run this example you must first download and install the following:

Bing Maps SDK: http://visualstudiogallery.msdn.microsoft.com/224eb93a-ebc4-46ba-9be7-90ee777ad9e1

You may need to update the project's reference to the Bing Maps SDK:
  1. Expand References under the HereMyAm project in Solution Explorer.
  2. If the Bing Maps reference is flagged (with a !) right click that entry and select Remove.
  3. Right click References and select Add References
  4. In the dialog box under Windows > Extensions, check the box for the Bing Maps SDK and click OK.
