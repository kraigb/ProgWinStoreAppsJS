This example builds a WinJS.Binding.List on top of an array of simple random numbers, optionally using the binding and proxy constructor options. The raw array as well as the List are shown in the output.

The two buttons change values in the array and the List, showing the relationship between the two depending on the options:

  • Binding will have no effect in this scenario because the array contains simple (nonobject) values.
  • Proxy will update the array whenever the List is modified. The List, however, does not update when the array changes and changing the array independently should be avoided.

Checking either checkbox will recreate the List from the current array.

Note that the output is just generated manually in both cases; data binding is not used with the List in this scenario.
