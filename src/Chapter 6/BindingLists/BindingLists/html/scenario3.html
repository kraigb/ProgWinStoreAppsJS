This example builds a WinJS.Binding.List from scratch using its push method, so there's no underlying array. Each item contains a random number from 1-100 and a random blue color value.

It then creates different projections of that same list using createFiltered, createSorted, createGrouped, and combinations of these.

The Modify List button repopulates the original List to show that the projections automatically update. The Modify Projection button changes properties of the first item in the grouped+sorted projection, showing that the change propagates to the underlying List as well as the projections.
