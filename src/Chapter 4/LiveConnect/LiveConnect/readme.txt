To run this example you must first download and install the following:

Live SDK 5.5: http://www.microsoft.com/en-us/download/details.aspx?id=40739

You may need to update the project's reference to the Live SDK:
  1. Expand References under the LiveConnect project in Solution Explorer.
  2. If the Live SDK reference is flagged (with a !) right click that entry and select Remove.
  3. Right click References and select Add References
  4. In the dialog box under Windows > Extensions, check the box for the Live SDK and click OK.
