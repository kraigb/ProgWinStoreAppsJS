Simple View Projection: Primary View

If projection is available, tap the Start button to create a view on the second monitor. If this button is disabled, a second monitor is not available. By handling the projectiondisplayavailablechanged event, though, it will be enabled if a second monitor is attached.

When a projection is active, use the Swap button to swap monitors and the Stop button to stop the projection.
